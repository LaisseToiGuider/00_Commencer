TP Numéro 5

Les Booléens et conditions multiples

Découvre Vrai ou Faux, et comment prendre des décisions complexes avec if, else if, et les opérateurs logiques !

Objectifs du TP

  • Comprendre les variables bool (true/false).
  • Utiliser les opérateurs logiques &&, ||, !.
  • Enchaîner des conditions avec else if.

Prérequis

  • Avoir fait le TP 1 (Les tests if/else).
  • Savoir utiliser Console.ReadLine() (TP 3).

Le type bool : Vrai ou Faux

En programmation, on a souvent besoin de savoir si quelque chose est vrai ou faux. C'est là qu'interviennent les booléens !

✅

Variables true et false

Un booléen (bool) ne peut contenir que deux valeurs : true (vrai) ou false (faux). 

bool estMajeur = true;
bool aLePermis = false;

if (estMajeur)
{
    Console.WriteLine("Tu es majeur !");
}

Les Opérateurs Logiques

Comment combiner plusieurs conditions ? Avec ET, OU, et NON !

&& (ET)

Il faut que toutes les conditions soient vraies.

A B A && B
false false false
false true false
true false false
true true true

|| (OU)

Il suffit qu'au moins une condition soit vraie.

A B A || B
false false false
false true true
true false true
true true true

! (NON)

Inverse la valeur : le vrai devient faux, et le faux devient vrai.

bool estFatigue = false;

if (!estFatigue) // Se lit "Si NON estFatigue"
{
    Console.WriteLine("On va jouer !");
}

Plusieurs choix avec else if

Parfois, on a plus de deux possibilités (pas juste if / else).

else if permet de tester une nouvelle condition si la précédente était fausse. 

int age = 15;

if (age < 12)
{
    Console.WriteLine("Tu es un enfant.");
}
else if (age < 18)
{
    Console.WriteLine("Tu es un ado.");
}
else
{
    Console.WriteLine("Tu es un adulte.");
}

À toi de jouer !

Pratique les conditions multiples et les booléens.

1
NIVEAU FACILE

Validation de mot de passe

Consigne :

Demande un mot de passe à l'utilisateur. Si le mot de passe est "secret123" ET que l'utilisateur tape "oui" pour confirmer, affiche "Accès autorisé". Sinon, affiche "Accès refusé".

Voir le corrigé 
Console.WriteLine("Entrez le mot de passe :");
string mdp = Console.ReadLine();

Console.WriteLine("Confirmez-vous ? (oui/non)");
string confirm = Console.ReadLine();

if (mdp == "secret123" && confirm == "oui")
{
    Console.WriteLine("Accès autorisé");
}
else
{
    Console.WriteLine("Accès refusé");
}
2
NIVEAU MOYEN

Menu simple

Consigne :

Affiche un menu avec 3 choix (1: Jouer, 2: Options, 3: Quitter). Demande à l'utilisateur de choisir un nombre. Utilise if / else if / else pour afficher un message correspondant au choix. Si le choix n'est pas 1, 2 ou 3, affiche "Choix invalide".

Voir le corrigé 
Console.WriteLine("Menu :");
Console.WriteLine("1: Jouer");
Console.WriteLine("2: Options");
Console.WriteLine("3: Quitter");
Console.WriteLine("Votre choix ?");

int choix = int.Parse(Console.ReadLine());

if (choix == 1)
{
    Console.WriteLine("Lancement du jeu...");
}
else if (choix == 2)
{
    Console.WriteLine("Ouverture des options...");
}
else if (choix == 3)
{
    Console.WriteLine("Au revoir !");
}
else
{
    Console.WriteLine("Choix invalide.");
}

Récapitulatif

  • bool : type de variable qui vaut true (vrai) ou false (faux).
  • && (ET) : Vrai si toutes les conditions sont vraies.
  • || (OU) : Vrai si au moins une condition est vraie.
  • ! (NON) : Inverse la valeur (vrai devient faux).
  • else if : Permet de tester une nouvelle condition si la précédente est fausse.
