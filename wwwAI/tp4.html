Les Boucles (do/while/for)

Comment répéter des instructions sans les réécrire ? Découvre la puissance de la répétition !

Objectifs du TP

  • Comprendre le principe d'une boucle.
  • Utiliser while et do...while.
  • Maîtriser la boucle for.

Prérequis

  • Avoir fait le TP 1 (Les tests if/else).
  • Savoir utiliser Console.ReadLine() (TP 3).

C'est quoi une boucle ?

Imagine que tu doives écrire "Bonjour" 100 fois. Tu ne vas pas copier-coller 100 lignes de code !

🔁

Le concept

Une boucle permet à l'ordinateur de répéter un bloc d'instructions tant qu'une condition est vraie.

while et do...while

Les boucles idéales quand on ne sait pas à l'avance combien de fois on va répéter.

while (Tant que)

Vérifie la condition avant d'entrer dans la boucle.

int compteur = 0;

while (compteur < 3)
{
    Console.WriteLine("Tour " + compteur);
    compteur++; // Ajoute 1
}

do...while (Faire...Tant que)

Exécute le code au moins une fois, puis vérifie la condition.

string motDePasse;

do
{
    Console.WriteLine("Mot de passe ?");
    motDePasse = Console.ReadLine();
} 
while (motDePasse != "secret");

La boucle for

La boucle la plus utilisée quand on sait exactement combien de fois on veut répéter.

Elle rassemble tout sur une seule ligne : le point de départ, la condition, et l'incrémentation (ajouter 1). 

// Pour i allant de 1 à 10 (inclus)
for (int i = 1; i <= 10; i++)
{
    Console.WriteLine("Valeur : " + i);
}
int i = 1

Départ

i <= 10

Condition

i++

À chaque tour (i = i + 1)

À toi de jouer !

Pratique les boucles avec ces exercices.

1
NIVEAU FACILE

Compter de 1 à 20

Consigne :

Utilise une boucle for pour afficher tous les nombres de 1 à 20 dans la console.

Voir le corrigé 
for (int i = 1; i <= 20; i++)
{
    Console.WriteLine(i);
}
2
NIVEAU MOYEN

Table de multiplication

Consigne :

Demande à l'utilisateur de saisir un nombre. Ensuite, utilise une boucle pour afficher la table de multiplication (de 1 à 10) de ce nombre.

Voir le corrigé 
Console.WriteLine("Entrez un nombre :");
int nombre = int.Parse(Console.ReadLine());

for (int i = 1; i <= 10; i++)
{
    Console.WriteLine(nombre + " x " + i + " = " + (nombre * i));
}
DÉFI BONUS

Le juste prix

Consigne :

L'ordinateur a un nombre mystère (par exemple 42). Demande à l'utilisateur de deviner ce nombre. Tant qu'il n'a pas trouvé, dis-lui "C'est plus !" ou "C'est moins !". S'il trouve, affiche "Bravo !". Utilise une boucle do...while ou while.

Voir la solution bonus 
int nombreMystere = 42;
int essai;

do
{
    Console.WriteLine("Devinez le nombre :");
    essai = int.Parse(Console.ReadLine());

    if (essai < nombreMystere)
    {
        Console.WriteLine("C'est plus !");
    }
    else if (essai > nombreMystere)
    {
        Console.WriteLine("C'est moins !");
    }

} while (essai != nombreMystere);

Console.WriteLine("Bravo ! Vous avez trouvé.");

Récapitulatif

  • Boucle : Répète un bloc de code tant qu'une condition est vraie.
  • while : Vérifie la condition avant d'exécuter.
  • do...while : Exécute au moins une fois, puis vérifie la condition.
  • for : Idéal quand on connaît le nombre exact de répétitions.
