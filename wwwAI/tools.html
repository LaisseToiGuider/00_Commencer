Installation des outils de code

Suis ces étapes pour préparer ton ordinateur au C#.

1

Installer VS Code

Visual Studio Code est l'éditeur de texte où nous allons écrire nos programmes. C'est comme un Word, mais pour le code !

Télécharger VS Code
2

Installer le SDK .NET

Le SDK .NET contient le compilateur (le traducteur) pour le langage C#. C'est lui qui va transformer ton code en application.

Télécharger .NET
3

Extension C#

Ouvre VS Code, va dans les extensions (les petits carrés à gauche) et cherche "C# Dev Kit". Installe-la !

Prêt à coder !

Si ça ne marche pas... (Dépannage)

L'informatique, ça bug parfois ! Voici comment résoudre les problèmes les plus fréquents :

"La commande dotnet n'est pas reconnue"

Redémarre ton ordinateur ou ferme complètement VS Code et rouvre-le. L'ordinateur a besoin de se mettre à jour après l'installation.

Le code est souligné en rouge

Vérifie que tu n'as pas oublié un point-virgule ; à la fin de la ligne, ou que tu as bien mis les majuscules aux bons endroits (C# est sensible à la casse).

Tout est installé ?

Génial ! Tu es maintenant prêt(e) à écrire ton tout premier programme. Allons-y !
